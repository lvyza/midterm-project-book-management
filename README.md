# Book Management Laravel Project

## Description
This is a **Book Management System** built with Laravel. It allows users to **Create, Read, Update, and Delete (CRUD)** books.

## Database Table: `books`
The table has the following fields:

| Field           | Type        | Description                |
|-----------------|------------|---------------------------|
| id              | integer    | Primary key               |
| title           | string     | Book title                |
| author          | string     | Author name               |
| isbn            | string     | Unique ISBN of the book   |
| published_year  | year       | Year the book was published |
| created_at      | timestamp  | Record creation time      |
| updated_at      | timestamp  | Record update time        |
